Selamat datang di dokumentasi resmi API Kalender Hijriah Global Tunggal (KHGT) Muhammadiyah. Layanan API ini dirancang untuk memudahkan developer dalam mengintegrasikan penanggalan Hijriah yang akurat, lengkap dengan data Masehi dan hari Pasaran Jawa ke dalam aplikasi web maupun mobile.

üöÄ Fitur Utama

  • Data JSON Akurat: Konversi format JSON yang mudah digunakan dari sumber KHGT resmi.
  • Pasaran Jawa Terpadu: Menyediakan data hari pasaran (Legi, Pahing, Pon, Wage, Kliwon).
  • Multi-Bahasa: Dukungan lokalisasi untuk berbagai bahasa.
  • Ringan & Cepat: API Native PHP yang dapat di-deploy di berbagai environment.

üåê Endpoint API

Gunakan parameter berikut untuk mengambil data kalender:

1. Berdasarkan Tahun (Hijriah)

GET https://domain-anda.com/api.php?year=1448

Contoh Response Data:

{
  "status": "success",
  "year": "1448",
  "data": [
    {
      "month_hijri": "Muharram 1448 H",
      "date_gregorian": "16 Juni 2026",
      "date_hijri": "1 Muharram 1448",
      "pasaran": "Selasa Legi"
    },
    ...
  ]
}

2. Berdasarkan Tanggal Spesifik (Masehi)

GET https://domain-anda.com/api.php?date=2026-02-18

Parameter date menerima format YYYY-MM-DD atau today.
